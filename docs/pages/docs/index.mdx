## Introduction

DB Lens is a modern and simplified database explorer tool that allows you to
view and manage your databases in a more intuitive way. It is designed to be a
lightweight and easy-to-use tool that can be used by anyone, regardless of their
technical background.

## Features

With DB Lens, you can easily connect to your databases,

- Explore Table Data
- Visualise Entity Relationship Diagrams
- Get Disk Usage
- Analyze Records, Indexes, Unused Indexes
- Manage Extensions (Available and Installed)
